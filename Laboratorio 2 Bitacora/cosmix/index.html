Primera Semana

Tuvimos la oportunidad de conocer a nuestro profesor y aprender más sobre él. Cada uno de nosotros se presentó de manera individual, compartiendo un poco acerca de quiénes somos. Posteriormente, comenzamos con la explicación detallada del sílabo del curso. Fue un buen comienzo para familiarizarnos con el profesor y tener una comprensión clara de lo que podemos esperar en el curso.

Segunda Semana

Procedimos a instalar Visual Studio Code, una herramienta esencial para el desarrollo web. Durante esta semana, también exploramos una página web existente para comprender su estructura y funcionamiento. Aprendimos conceptos fundamentales del lenguaje HTML, como las partes de un elemento, el uso de Doctype, la sección Head y la etiqueta Title, que son elementos clave en la construcción de páginas web. Después de adquirir esta base teórica, nos sumergimos en la práctica. Iniciamos Visual Studio Code y comenzamos a editar un código paso a paso siguiendo las indicaciones detalladas del profesor. Esta experiencia nos permitió aplicar los conocimientos recién adquiridos y comenzar a familiarizarnos con la creación y edición de contenido web.Pusimos a prueba los conocimientos que hemos adquirido hasta el momento utilizando la plataforma Quizizz. Esta actividad nos permitió evaluar nuestro progreso y comprensión de los temas anteriores. Luego, nos sumergimos en el mundo de la creación de una bitácora digital. Aprendimos cómo crear y estructurar una bitácora en línea, y cómo agregar enlaces y vínculos a nuestros textos.

Tercera Semana

En el transcurso de esta semana, hemos adquirido habilidades para realzar y dar formato al contenido de nuestra bitácora . Una de las principales lecciones que hemos abordado es la capacidad de enfatizar palabras o frases mediante el uso de las etiquetas (negrita) y (cursiva). Estas etiquetas, tienen el poder de resaltar y comunicar de manera efectiva nuestras ideas en la web. Tambien trabajamos los atributos adiconales y mas etiquetas como apariencia de caracter y apariencia de espacio. Además de esto, hemos puesto en práctica la habilidad de ajustar el tamaño de las imágenes, permitiéndonos perfeccionar aún más la presentación visual de nuestro contenido. Hablamos tambien sobre la Semántica, esta nos ayuda a que la gente pueda llegar hasta nuestro contenido según los enlaces que coloquemos seran denominadas como palabras claves sobre nuestro tema.

Cuarta Semana

Durante esta Semana de clase, hemos adentrado en el mundo del CSS. El CSS nos brinda una herramienta esencial para dar forma y personalidad a los elementos HTML en nuestras páginas web. Con él, hemos adquirido la capacidad de crear reglas que detall la apariencia y el estilo de nuestros contenidos. Desde la elección de fuentes y tamaños de texto hasta la paleta de colores y la disposición de elementos en la página, el CSS se ha convertido en una herramienta fundamental en nuestro diseño web, ya que nos permite diseñar a nuestro gusto.

Quinta Semana

Durante esta semana de aprendizaje, hemos explorado dos aspectos fundamentales en el mundo del diseño web: el juego de colores y transparencias aplicado al texto y la capacidad de modificar el tamaño y disposición de nuestras cajas (o "Boxes"), los elementos esenciales que albergan nuestro contenido web. Con un enfoque especial en las propiedades como "Height" para definir la altura y "Width" para establecer el ancho de estas cajas, "Border", "Margin" y "Padding". Esta semana en nuestra clase presencial tambien aprendimos sobre DNS, que es el sistema de nombres de dominio, Cada dispositivo conectado a Internet tiene una dirección IP única que otros equipos pueden usar para encontrarlo.
